# Draft Review Payload Template

Use this shape when calling `review_outreach_draft`.

```json
{
  "campaignBrief": "",
  "audienceSummary": "",
  "draftMessage": "",
  "mediaReference": null
}
```

Checklist before calling:

- Campaign intent is clear.
- Draft is customer-facing.
- Draft contains no invented claims.
- Draft is short enough for WhatsApp.
- Audience summary includes valid and skipped counts if known.
